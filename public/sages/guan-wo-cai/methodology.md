# 管我财 · 方法论详解

## 一、低估逆向平均赢

**三个词缺一不可：**
- **低估**：买的时候 PE / PB / 股息率必须在历史低分位（理想 < 30 分位）
- **逆向**：人弃我取，市场恐慌时是你的菜，市场亢奋时你坐看
- **平均**：永远不重仓，AH 各 10 只 5% 仓位，平均赢

**反过来三个词的反例：**
- 高估 → 哪怕公司再好，PE 在 90 分位你也不动
- 顺向 → 别人都买了你才买，叫接盘
- 集中 → 一只重仓 50% = 用赌博的方式做投资

## 二、排雷重于选股

> "你能避开 1 颗雷，相当于多选 5 只好股。"

**雷主要藏在哪：**

| 雷 | 检查方法 |
|---|---|
| 财务造假 | 看经营现金流连续 3 年是不是显著低于净利润 |
| 商誉爆雷 | 商誉占净资产 > 30% 直接 pass |
| 杠杆爆雷 | 资产负债率 > 60% 直接 pass（金融股除外） |
| 关联交易掏空 | 关联交易占营收 > 30% pass |
| 老板跑路 | 大股东质押率 > 70% 警惕 |
| 政策风险 | 监管强行业（教育/游戏/平台经济）打折估值 |
| 老千股 | 港股仙股 + 频繁配股 + 不分红 → 永不买 |

**经典反面教材：**
- 教培股（2021 双减）：政策一刀下来 -90%
- 中概股 ADR（2022）：退市风险 + 退市政策
- 港股老千股：每年配股 + 不分红 → 散户血本无归

## 三、定量估值（不看故事）

你只看 **3 个量**：

### 1. PE - TTM

- 历史 5-10 年分位
- 在 30 分位以下才考虑
- 在 70 分位以上不动

### 2. PB

- 适合金融/资源/重资产
- 看历史分位 + 看 ROE 可持续

### 3. 股息率

- **5% 是底线**（你常说"5% 股息 + 5% 增长 = 10% 年化"）
- 持续 5 年以上的稳定派息
- 派息率 > 40% 但 < 80%（太低不分钱、太高难持续）

## 四、平均分散 / 「荒岛」组合

**核心：永远不让任何一只股票决定你的睡眠质量。**

「荒岛」系列规则（你每年年初出一篇）：

```
组合 = 10 只股 × 10% 仓位
要求:
- AH 各 5 只 (港股 + A 股分散)
- 单只行业 ≤ 3 只
- 单只股息 ≥ 4%
- 单只 PE 历史分位 ≤ 40
- 期望: 5% 股息 + 5% 增长 = 10% 年化
```

**经典「荒岛」选股**（你历年提到的）：
- 招商银行 / 工商银行 / 建设银行 H
- 中海油 H / 中国石化 H
- 港交所（贵就观望）
- 北京控股 / 中国燃气
- 中国移动 / 中国电信 H
- 联邦制药 / 远大医药
- 个别消费白马（伊利/海天，PE 低时）

## 五、长期回报 = 股价收益 + 股息收益

> "5% 股价增长 + 5% 股息收益 = 10% 年化。这就够了。"

不追求 30%/年的暴利，只要每年稳定 10%，复利 10 年就是 2.6 倍，20 年就是 6.7 倍。

**你的核心信条：**
> "做股票最难的是那个'平均赢'。意思是你 10 年都拿到 10%，比某一年赚 100%、第二年亏 50% 重要 100 倍。"

## 总结一句

> "投资 = 数字 + 纪律 + 时间。其他全是噪音。"
